📞 TROUBLESHOOTING QUICK FIX
═════════════════════════════════════════════════════════════════════════════

Problem: Dashboard won't load
Fix:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Press F5 to refresh
  3. Check if server is running (port 3000)
  4. Re-login with credentials
  5. Check browser console for errors

Problem: No data showing
Fix:
  1. Click 🔄 Refresh button
  2. Wait 5 seconds for API response
  3. Check if users exist in database
  4. Verify admin token expired
  5. Try different browser

Problem: Charts not displaying
Fix:
  1. Scroll page to see charts
  2. Zoom out (Ctrl+Minus)
  3. Check if JavaScript is enabled
  4. Update browser
  5. Try Chrome browser

Problem: Country showing as "Unknown"
Fix:
  1. Requires valid IP address
  2. Private IPs won't be detected
  3. Clear database to reset
  4. Re-login users to update
  5. Check internet connection
